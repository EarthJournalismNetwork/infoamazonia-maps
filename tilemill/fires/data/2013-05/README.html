===============================================
README: LANCE FIRMS MODIS Fire/Hotspot Data Download
===============================================

This zip file supplied to you contains active fire data for the requested dates in your area-of-interest. The data may include fire detections from MODAPS, the definitive version of Collection 5 (version 5.1), and from LANCE Rapid Response [RR] (version 5.0). The "version" field in the attached data indicates the source. Please refer to the following FAQ for more information for the difference between MODAPS and MRR at http://earthdata.nasa.gov/data/nrt-data/help/faq#firms35

Please note: If your request results in no fire points, the accompanying ZIP file will include an empty CSV file with a header, or an empty DBF file. If you believe that this has occurred due to an error, please contact us at support@earthdata.nasa.gov

For more information regarding the attribute fields, please visit the following page: http://earthdata.nasa.gov/data/nrt-data/help/faq#firms20. Visit the FIRMS project website at http://earthdata.nasa.gov/data/nrt-data/firms

======================
PROJECTION INFORMATION
======================

The MODIS hotspot data supplied to you are in the WGS-84 geographic projection (the "latitude/longitude projection").For help on importing the data into your local GIS, please refer to our FAQ at http://earthdata.nasa.gov/data/nrt-data/help/faq.

==========
IMPORTANT
==========

For further information, please refer to the MODIS Fire Users Guide (http://earthdata.nasa.gov/sites/default/files/field/document/MODIS_Fire_users_Guide_2.4.pdf) and the FAQ section (http://earthdata.nasa.gov/data/nrt-data/help/faq).

Please note that there is data missing from several of the data sets. There is data missing from end of June to the beginning of July in 2001, 2002 is missing some data throughout the data set, 2007 has some missing data from mid August and data is missing for part of 21 April 2009, and missing for 22 April 2009. There might also be some erroneous data present in the data set. Please refer to the disclaimer below.
